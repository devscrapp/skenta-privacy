Skenta collecte uniquement les données strictement nécessaires au fonctionnement du service. Ce document décrit de manière exhaustive la nature des données traitées, leur finalité et vos droits.

Données collectées et finalités

Localisation GPS — Utilisation au premier plan uniquement

Votre position géographique est utilisée pour identifier les stations-service proches de vous et calculer les distances. Elle n'est jamais stockée sur nos serveurs.

Base légale : Intérêt légitime — fourniture d'un service de proximité (Article 6.1.f du RGPD)

Quand ? Uniquement lorsque vous utilisez activement Skenta (application au premier plan). Skenta n'accède jamais à votre localisation en arrière-plan.

Comment ? Les coordonnées GPS sont transmises à notre serveur pour la recherche de stations proches, puis supprimées immédiatement. Aucun historique de localisation n'est conservé.

Partage ? Votre localisation n'est jamais partagée avec des tiers à des fins publicitaires ou de profilage.

Token de notification (FCM)

Pour l'envoi d'alertes de prix, l'application utilise Firebase Cloud Messaging (FCM), un service de Google LLC. Un identifiant technique anonyme est généré par votre appareil lors de l'activation des notifications.

Base légale : Consentement — vous autorisez explicitement les notifications (Article 6.1.a du RGPD)

Ce token est propre à l'installation de l'application sur votre appareil. Il ne permet pas de vous identifier personnellement.

Durée de conservation : Le token est supprimé automatiquement dans les cas suivants :

Immédiatement lors de la désinstallation de l'application

Immédiatement via Paramètres → Effacer mes données

Après 180 jours par les serveurs Firebase en cas d'inactivité

Vous pouvez demander la suppression manuelle du token à tout moment via contact@skenta.app.

Firebase Cloud Messaging est opéré par Google LLC (États-Unis). Le token peut transiter vers des serveurs situés hors de l'Union Européenne, dans le cadre des clauses contractuelles types approuvées par la Commission Européenne (Article 46 du RGPD).

Données stockées localement sur votre appareil

Les données suivantes sont conservées exclusivement sur votre appareil et ne transitent à aucun moment par nos serveurs :

Véhicules enregistrés (nom, type, carburant, consommation moyenne)

Stations favorites

Préférences de l'application (carburant par défaut, rayon de recherche, ordre de tri)

Cache des tuiles cartographiques (OpenStreetMap)

Ces données sont supprimées définitivement lors de la désinstallation de l'application.

Signalements de prix

Lorsque vous signalez un prix incorrect, les informations suivantes sont transmises à notre serveur :

L'identifiant de la station, le type de carburant et le prix signalé

Un condensat anonyme (hash non réversible) de votre appareil, utilisé uniquement à des fins anti-spam

Ces données servent exclusivement à valider la fiabilité des prix par consensus entre utilisateurs. Aucun identifiant personnel n'est associé à un signalement.

Publicité

Google AdMob

Skenta est une application gratuite financée par la publicité. Nous utilisons Google AdMob, un service publicitaire opéré par Google LLC (États-Unis), pour diffuser des annonces.

Base légale : Consentement — un formulaire de consentement conforme IAB TCF est présenté au premier lancement (Article 6.1.a du RGPD)

Google AdMob peut utiliser l'identifiant publicitaire Android (GAID) pour personnaliser les annonces, sous réserve de votre consentement.

Les données publicitaires peuvent être transférées vers des serveurs de Google LLC situés aux États-Unis, encadrées par les clauses contractuelles types de la Commission Européenne.

Vous pouvez modifier ou retirer votre consentement à tout moment via Paramètres → Gérer mes préférences publicitaires.

Sécurité des données

Protection des données en transit

Toutes les transmissions de données entre Skenta et nos serveurs sont chiffrées via le protocole HTTPS (Transport Layer Security — TLS 1.2 ou supérieur).

Vos données ne sont jamais transmises en clair sur le réseau.

Les données sensibles (localisation, token FCM) sont supprimées des serveurs dès que le traitement est terminé.

Bien que nous prenions toutes les mesures raisonnables pour sécuriser vos données, aucune transmission par Internet n'est garantie à 100% sécurisée.

Vos droits

Vos droits

Skenta ne collecte aucune donnée personnelle identifiable (nom, email, compte utilisateur). Les seules données liées à votre appareil sont le token FCM (anonyme, si vous activez les notifications) et les signalements de prix (anonymes).

Pour ces données, vous disposez des droits suivants :

Retrait du consentement publicitaire : via Paramètres → Gérer mes préférences publicitaires à tout moment

Réclamation auprès de la CNIL : cnil.fr

Dispositions particulières

Protection des mineurs

Skenta n'est pas destinée aux personnes de moins de 16 ans et ne collecte pas sciemment de données les concernant. Si vous êtes le représentant légal d'un mineur et pensez que celui-ci a utilisé l'application, veuillez nous contacter à contact@skenta.app afin que nous procédions à la suppression des données associées.
