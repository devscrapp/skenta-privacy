Politique de confidentialité

Skenta — Dernière mise à jour : 9 avril 2026

1. Présentation

Skenta est une application mobile gratuite affichant les prix des carburants en temps réel en France. Elle est développée et maintenue de manière indépendante.

2. Données collectées

Position géographique

Votre position GPS est utilisée uniquement pour afficher les stations proches de vous et calculer les distances. Elle n'est jamais enregistrée sur nos serveurs ni transmise à des tiers.

Identifiant de notification (FCM Token)

Un identifiant anonyme est transmis à notre serveur uniquement pour vous envoyer vos alertes de prix personnalisées. Il ne contient aucune donnée personnelle identifiable.

Données stockées localement

Vos véhicules, stations favorites, alertes et préférences sont enregistrés uniquement sur votre appareil. Ces données ne transitent jamais par nos serveurs.

3. Publicité

Skenta affiche des publicités pour financer son fonctionnement gratuit. Ces publicités sont diffusées par un partenaire certifié conforme au RGPD.

Lors de votre premier lancement, vous êtes invité à choisir vos préférences publicitaires. Vous pouvez modifier ce choix à tout moment dans Paramètres → Préférences publicitaires.

4. Données transmises à notre serveur

Notre serveur reçoit uniquement :

  • Votre position GPS approximative pour rechercher les stations proches — non enregistrée
  • Votre FCM Token pour l'envoi d'alertes de prix — stocké de manière sécurisée
  • Vos signalements de prix — anonymes

Aucune donnée personnelle identifiable (nom, email, téléphone) n'est collectée.

5. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : savoir quelles données nous détenons
  • Droit de suppression : supprimer votre FCM Token via Paramètres → Effacer toutes les données
  • Droit de retrait du consentement : modifier vos préférences publicitaires à tout moment

Pour toute demande : contact@skenta.app

6. Conservation des données

  • FCM Token : supprimé automatiquement si aucune alerte active depuis 90 jours
  • Alertes de prix : expiration automatique après 90 jours
  • Signalements : anonymes, conservés pour améliorer la qualité des données

7. Contact

Pour toute question relative à vos données personnelles :
contact@skenta.app
